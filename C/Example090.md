# 练习实例90

## 题目

利用指针倒序输出数组。


## 分析

本题考查的知识点：
- 数组
- 指针
- while 和 for 循环语句

分析：
- 其实考查的就是利用指针来访问和修改元素，交换数组中 i 和 M-1-i 对应下标的元素，交换 M/2 次，最后就完成了数组元素的逆序。

## 代码

```c
#include <stdio.h>

#define M 5

int main() {
    int a[M] = {1, 2, 3, 4, 5};
    int i, j, t;

    // 交换数组中 i 和 M-1-i 对应下标的值
    i = 0;
    j = M - 1;
    while (i <= j) {
        t = *(a + i);
        *(a + i) = *(a + j);
        *(a + j) = t;
        i++;
        j--;
    }

    // 打印交换后的数组元素
    for (i = 0; i < M; i++) {
        printf("%d\t", *(a + i));
    }
}
```

代码执行结果如下：

```text
5       4       3       2       1
```
