# 练习实例55

## 题目

学习使用按位取反~。


## 分析

所谓的按位取反，即二进制如果是0则取为1，如果是1则取为0。

但注意，在C语言中二进制是以补码形式存放的，要转换成十进制数也需要先转换成原码。

## 代码

```c
#include <stdio.h>

int main() {
    int a = 3;// 它的二进制是 0000 0011
    int b = ~a;// 按位取反，它的结果是 1111 1100
    printf("%d", b);// -4，注意结果为-4，跟二进制补码有关
}
```

代码执行结果如下：

```text
-4
```
