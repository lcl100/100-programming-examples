# 练习实例1
## 题目
有1、2、3、4个数字，能组成多少个互不相同且无重复数字的三位数？都是多少？

## 分析
三位数表示要三层循环，这里使用for循环比较方便。其中循环变量的范围是[1,4]，并且要求三位数的个位、十位、百位互不相同，因此要排除掉循环变量中相同的情况。

## 代码
```c
#include <stdio.h>

int main() {
    // 三层for循环
    for (int x = 1; x <= 4; x++) {
        for (int y = 1; y <= 4; y++) {
            for (int z = 1; z <= 4; z++) {
                // 确保三个变量互不相同
                if (x != y && y != z && x != z) {
                    printf("%d,%d,%d\n", x, y, z);
                }
            }
        }
    }
}
```
