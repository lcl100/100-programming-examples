# 练习实例124

## 题目

输入一个华氏温度，输出摄氏温度，计算公式为`c=5/9*(F-32)`。要求结果保留两位小数。


## 分析

本题考查的知识点：
- 浮点数
- `printf`和`scanf`函数

分析：
- 本题很简单，利用公式直接转换即可。
- 但注意公式中的`5/9`在 C 语言中会直接等于 0，所以调整公式顺序为：`c=5*(F-32)/9`。

## 代码

```c
#include<stdio.h>

int main() {
    // 从键盘输入温度
    float temp;
    printf("请输入一个华氏温度：");
    scanf("%f", &temp);

    // 根据公式转换为摄氏温度
    float c = 5 * (temp - 32) / 9;// 注意交换下顺序，因为 5/9=0

    // 打印结果
    printf("%.2f", c);
}
```

代码执行结果如下：

```text
请输入一个华氏温度：15
-9.44
```
