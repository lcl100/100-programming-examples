# 练习实例16

## 题目

输入两个正整数m和n，求其最大公约数和最小公倍数。


## 分析

分析：
- 本题偏向于数学。
- 最小公倍数 = 输入的两个数之积除于它们的最大公约数，关键是求出最大公约数。
- 求最大公约数用辗转相除法（又名欧几里德算法）。


## 代码

```c
#include <stdio.h>

int main() {
    // 从键盘输入两个整数
    int a, b;
    printf("请输入两个整数：");
    scanf("%d%d", &a, &b);

    int i = a, j = b;
    int temp;// 充当两个变量交换的媒介
    if (a < b) {
        temp = a;
        a = b;
        b = temp;
    }
    while (b != 0) {
        temp = a % b;
        a = b;
        b = temp;
    }
    // 最小公倍数
    int t = i * j / a;
    // 打印结果
    printf("%d 和 %d 的最大公约数是：%d\n", i, j, a);
    printf("%d 和 %d 的最小公倍数是：%d\n", i, j, t);
}
```
